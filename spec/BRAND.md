# molt | The "Double Shell"
This represents the migration process. It shows one shell being left behind (the ghosted outline) and a new, solid shell being inhabited (the migration target).

## Design logic:
- **The Visual**: Two overlapping carapaces (lobster shells).
- **The Transition**: The dashed line represents the "source" or legacy environment, while the solid red represents the "destination."
- **The Core**: A small blue circle represents the "Agent" moving between them.
